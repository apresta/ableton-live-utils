**EditAudioWith.app** (macOS-only):

Set this as your sample editor in Live's Preferences. Upon editing a sample, it will pop up a menu, letting you choose among different editing apps (RX, Melodyne, and Audacity).\

You can customize the app menu by opening this app with **Script Editor**.

***

**MackieControl_PreventFaderFocus**:

Modified remote script for MackieControl surfaces. It removes the default behavior where interacting with a fader causes the corresponding track to be focused in Live. Useful if you want your faders to only do one thing (change volume), without interfering with other controllers or keyboard/mouse.

Copy the folder inside **User Library/Remote Scripts** and configure your control surface to use the script in Live's Preferences.
